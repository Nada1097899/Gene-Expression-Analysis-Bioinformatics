#  Differential Gene Expression Analysis

A bioinformatics project focused on analyzing genomic data to identify significant genes associated with specific biological conditions or diseases.

## Project Overview
This project involves processing complex gene expression datasets to discover which genes are "up-regulated" or "down-regulated" under certain conditions. By applying statistical filters (P-values and Fold Change), we can identify potential biomarkers for clinical research.

##  Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scipy (Stats).
* **Key Techniques:** Data Merging (Metadata + Expression), Log2 Fold Change calculation, and Volcano Plotting.

## Methodology
- **Data Integration:** Merging SDRF metadata with expression counts to align clinical attributes with genetic data.
- **Statistical Testing:** Applying T-tests to determine the significance of expression changes across different cohorts.
- **Visual Exploration:** Utilizing Volcano Plots to visualize the distribution of significant genes.

##  Key Findings
- **Top Significant Genes:** Identified the most affected genes (e.g., MAPK8IP1P2, TDRD1) based on their expression levels.
- **Data Quality:** Handled missing values and normalized gene names for better interpretability.
